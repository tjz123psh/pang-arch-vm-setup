# Web Search

通过 opencode 内置 `websearch` + `webfetch` 进行多策略搜索，按意图和语言优先级自动选择方案。

## 语言优先级

```
英文资料优先 → 英文结果不够 → 切中文查
```

- 始终先查英文关键词，除非用户明确指定查中文（如"用中文搜"、"找国内资料"）
- 英文结果不够（<3 条有效结果，或内容明显不相关）→ 用中文同义关键词重搜
- 中文结果也不够 → 调换关键词方向后重试

## 搜索策略

| 意图 | 策略 |
|------|------|
| 技术查证/对比 | `websearch(type=deep)` → `webfetch` 读原文 → 对比摘要 |
| 查官方文档 | `websearch("site:xxx " + query)` 精确域名限定 |
| 快速查定义 | `websearch(type=fast)` → 直接回摘要 |
| 多角度对比 | 同一 query 不同措辞搜 2-3 轮 → 合并去重 |
| 中文限定 | 用户明确要中文时，先中文搜索，不够再切英文 |

## 执行步骤

1. 判断搜索意图和是否需要英文优先
2. 用英文关键词 `websearch(type=deep, numResults=8)`
3. 评估结果：数量够？内容相关？来自权威源？
4. **英文结果不足** → 翻译成中文关键词重搜
5. 关键结果用 `webfetch` 读原文（优先英文原文，次选中译版）
6. 合并摘要输出，标注语言来源

## 注意事项

- 不重复已有的内置工具能力，只做策略编排
- deep 模式搜不到时切 fast 模式试一次，仍不行问用户要不要换方向
- webfetch 的 URL 取自搜索结果，不自行拼接
